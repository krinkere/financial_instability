Swing Index

The Swing Index is a technical indicator that attempts to predict future short-term price action:

  • When the Swing Index crosses over zero, then a trader might expect short-term price movement upward.
  • When the Swing Index crosses below zero, then a trader might expect short-term price movement downward.

Accumulative Swing Index

the Accumulative Swing Index (ASI) is mainly used as a divergence and confirmation tool, but can be used for buy and sell signals as well.

  • Potential Buy Signal - A trader might consider a buy signal when the Accumulative Swing Index breaks above a downward trendline.
  • Potential Sell Signal - A trader might consider a sell signal when the Accumulative Swing Index breaks below an upward trendline
